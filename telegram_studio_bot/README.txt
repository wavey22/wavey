Инструкция по запуску Telegram-бота студии звукозаписи:

1. Установите зависимости:
   pip install pyTelegramBotAPI

2. Вставьте ваш Telegram API токен в main.py вместо 'YOUR_BOT_TOKEN_HERE'.

3. Запустите бота:
   python3 main.py

Бот поддерживает команды:
- /start: запускает меню
- Кнопки: запись, прайс, портфолио, контакты, FAQ
